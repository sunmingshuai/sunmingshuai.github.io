    Spring源码分析一:环境搭建

    2017-04-15

    想研究spring源码的第一步当然是源码的搭建 推荐大家使用idea集成开发环境 下面的搭建过程也以idea为基础
    需要gradle git环境 不再叙述
    从github仓库中clone一份代码到本地
    git clone -b 3.2.x git@github.com:spring-projects/spring-framework.git
    这里需要注意的是你所使用的jdk版本与spring源码所需要的版本的匹配问题 如果你使用的jdk版本是1.7 而clone的是4以上的spring源码的话 那么comiple的时候就会报错
    查看源码目录下的 import-into-idea.md 文件 发现步骤可真简单
